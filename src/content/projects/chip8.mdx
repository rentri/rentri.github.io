# A chip8 emulator written in Golang

## Tech Used

- **[Go (Golang)](https://go.dev/)** – core emulator implementation
- **[Ebiten](https://ebitengine.org/)** – rendering, game loop, sound output and input handling

## Running the emulator

### Prerequistes
- go 1.24.4

### Clone the repositry
```bash
git clone https://github.com/rentri/goChip8.git
```

### Run & Build
```bash
cd goChip8
```
Run without building
```bash
go run . <path-to-rom>
```
OR

Build
```bash
go build -o chip8
```
Then run
```bash
./chip8 <path-to-rom>
```

  Flags test

  > Our implementation of SHR and SHL instructions uses modern beavior.
  > Hence 8XY6 & 8XYE fail the last check.
  
  ![Flags Test](/img/chip8/flags.png)
  
  Quirks test

  ![Quirks Test](/img/chip8/quirks.png)
    
  Keypad test

  > Some key presses may occasionally not register due to high CPU speed or keyboard polling.
  > If you press a key when the emulator was not checking if it is pressed, you will experience "key miss", where the press won't be registered.
  > The slower the emulation speed the higher chance for it to happen.
